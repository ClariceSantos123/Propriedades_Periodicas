# 🔬 Propriedades Periódicas Interativas

## ✨ Nova Versão Gamificada da Tabela Periódica

Sistema educacional interativo focado em ensinar as **7 principais propriedades periódicas** através de desafios gamificados e visualizações dinâmicas!

---

## 🎯 VISÃO GERAL

### **O que é?**

Uma evolução da Tabela Periódica Interativa, agora focada em:
- ✅ **Ensinar propriedades periódicas** (não apenas símbolos)

## 📚 AS 7 PROPRIEDADES TRABALHADAS

### **1. Raio Atômico** 📏
- O que é: Tamanho do átomo (pm)
- Tendência: ↑ descendo grupos, ↓ andando períodos
- Desafios: Comparar tamanhos, ordenar elementos

### **2. Eletronegatividade** ⚡
- O que é: Força para atrair elétrons
- Tendência: ↑ subindo grupos, ↑ andando períodos
- Desafios: Identificar mais eletronegativos, prever valores

### **3. Energia de Ionização** 💥
- O que é: Energia para remover elétron (kJ/mol)
- Tendência: ↑ subindo grupos, ↑ andando períodos
- Desafios: Comparar energias, explicar tendências

### **4. Afinidade Eletrônica** 🔋
- O que é: Energia ao ganhar elétron
- Tendência: Varia (halogênios têm alta)
- Desafios: Identificar elementos que querem elétrons

### **5. Caráter Metálico** ⚙️
- O que é: Comportamento como metal (0-10)
- Tendência: ↑ descendo grupos, ↓ andando períodos
- Desafios: Classificar metais/não-metais, linha divisória

### **6. Reatividade** 🧨
- O que é: Facilidade de reagir
- Tendência: Varia por família (alcalinos e halogênios reativos)
- Desafios: Comparar reatividades, prever reações

### **7. Densidade** ⚖️
- O que é: Massa por volume (g/cm³)
- Tendência: Aumenta no centro da tabela
- Desafios: Ordenar por densidade, identificar mais densos

## 🎮 TIPOS DE DESAFIOS

### **Tipo 1: "Maior ou Menor?"** ⚡ RÁPIDO
```
Mostra 2 elementos
Pergunta: "Qual tem MAIOR [propriedade]?"
Aluno clica na resposta
Feedback imediato + explicação
```

**Exemplo:**
```
Qual tem MAIOR raio atômico?

[ Na ]    vs    [ Cl ]
   ↑              ↓
  CLIQUE       CLIQUE

Resposta: Na (186 pm > 99 pm)
Explicação: "Sódio está mais à esquerda e abaixo!"
```

---

### **Tipo 2: "Ordenar 4 Elementos"** 📊 MÉDIO
```
"Ordene do MENOR para o MAIOR:"
[Arraste os cards]

Inicial: [ Cl ] [ K ] [ Na ] [ F ]
Correta: [ F ] [ Cl ] [ Na ] [ K ]

Sistema valida ordem
Mostra gráfico da tendência
```

---

### **Tipo 3: "Pintar a Tabela"** 🎨 DIFÍCIL
```
"Clique nos 5 elementos de MAIOR eletronegatividade"

[Tabela interativa 18×7]
Aluno clica em células
Sistema compara com resposta (F, O, Cl, N, Br)
Mostra mapa de calor correto
```

---

### **Tipo 4: "Prever o Valor"** 🔮 AVANÇADO
```
"O elemento está em Grupo 2, Período 4"
"Qual será seu raio atômico aproximado?"

( ) 100 pm
(✓) 197 pm  ← CORRETO!
( ) 250 pm

Explicação: "Grupo 2, período 4 = Ca = ~197 pm"
```

---

### **Tipo 5: "Corrida da Tendência"** 🏃 DIFÍCIL
```
Animação mostra propriedade mudando:
Li → Na → K → Rb → Cs → Fr

PARE quando atingir o MÁXIMO!

[Barra crescendo em tempo real]
Aluno precisa clicar no momento certo
Pontuação baseada em precisão
```

## 🏆 SISTEMA DE PROGRESSÃO

### **Níveis por Propriedade:**
```
Nível 1: Fácil    → ⭐ (10 perguntas certas)
Nível 2: Médio    → ⭐ (8 perguntas certas)
Nível 3: Difícil  → ⭐ (6 perguntas certas)
Nível 4: Expert   → ⭐ (4 perguntas certas)
Nível 5: Mestre   → ⭐ (Desafio especial!)
```

### **Pontuação:**
```
Acerto rápido:   +150 pontos
Acerto normal:   +100 pontos
Acerto com dica: +50 pontos
Erro:            -10 pontos
```

### **Conquistas:**
```
🏆 "Mestre do Raio Atômico" - 5 estrelas no Raio
⚡ "Guru da Eletronegatividade" - 5 estrelas
💎 "Químico Completo" - 35 estrelas (todas)
🎯 "Perfeccionista" - 100% de acerto em 1 propriedade
⏱️ "Velocista" - Completar nível em < 2min
```

## 💡 PEDAGOGIA E DESIGN

### **Princípios Seguidos:**

#### **1. Aprendizagem Ativa**
- Aluno INTERAGE (não apenas lê)
- Descobre padrões sozinho
- Constrói conhecimento através da prática

#### **2. Scaffolding (Suporte Gradual)**
- Começa fácil (comparar 2 elementos)
- Aumenta complexidade (ordenar 4)
- Termina avançado (prever valores)

#### **3. Feedback Imediato**
- Sabe se acertou instantaneamente
- Vê explicação do porquê
- Visualiza a tendência correta

#### **4. Gamificação Não Punitiva**
- Erros custam apenas -10 pontos
- Sempre pode tentar novamente
- Progresso é salvo (LocalStorage)

#### **5. Múltiplas Representações**
- Numérica (valores exatos)
- Visual (cores, gráficos)
- Textual (explicações)
- Interativa (manipulação)

## 🎓 FUNDAMENTAÇÃO TEÓRICA

### **Teorias Aplicadas:**

#### **1. Teoria da Aprendizagem Significativa (Ausubel)**
- Conecta nova info (tendências) com prévia (tabela)
- Subsunçores: "elementos do mesmo grupo têm..."
- Diferenciação progressiva: fácil → difícil

#### **2. Construcionismo (Papert)**
- Aluno constrói conhecimento ativamente
- Manipula elementos, testa hipóteses
- Aprende fazendo (learning by doing)

#### **3. Carga Cognitiva (Sweller)**
- Informação dividida em 7 propriedades (não 118×7)
- Visual ajuda (mapa calor) reduz carga intrínseca
- Desafios progressivos evitam sobrecarga

#### **4. Gamificação Educacional (Kapp, 2012)**
- Elementos de jogos (pontos, níveis, conquistas)
- MAS: Hanus & Fox (2015) - não punitiva!
- Motivação intrínseca > extrínseca
